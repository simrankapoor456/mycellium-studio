# Mycellium Studio — Logo Exploration 01

This folder is a standalone identity study. It does not replace or modify any application or production brand asset.

Open `comparison.html` in a browser for the full interactive comparison, scorecard, optical-size tests, dark/light/monochrome previews, and editable SVG downloads.

## Files

- `comparison.html` — interactive comparison page
- `wordmark.svg` — shared editable editorial wordmark
- `01-living-network/` through `06-asymmetric-growth/` — master icon, simplified 16 px icon, and icon-plus-wordmark lockup for each direction
- `raster-tests/` — generated PNG checks at 512, 128, 64, 32, 24, and 16 px
- `render-raster-tests.mjs` — reproducible local renderer for the PNG checks

## Palette

- Near-black forest: `#08110E`
- Deep pine: `#123229`
- Warm cream: `#F2EBDD`
- Bioluminescent lime: `#C7F36B`
- Restrained antique gold: `#A58B52`

All icon masters use `currentColor`. Lockups remain one-color and editable. The type is preserved as SVG text so the wording, tracking, and font stack can be edited; convert text to outlines only at final production handoff after a typeface is licensed and selected.

## Comparison note

Scores are comparative prompts, not a winner selection. “Generic AI risk” is the only inverse metric: lower is better.
